# iOS To-Do App
Simple application for creating a list of To-Do's. Includes Core Data to persist your to-do list after the app closes.

## Users
- Can create to-do's
- Can mark to-do's with as accomplished with a check mark
- Can delete to-do's

## Technologies
- **Swift iOS**
